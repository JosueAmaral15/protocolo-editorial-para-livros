# Build

Esta pasta guarda intermediarios reconstruiveis.

Nao use `build/` como fonte principal do livro.

- `tmp/`: arquivos temporarios.
- `chapters/`: capitulos extraidos ou separados por automacao.
- `exports/`: exportacoes intermediarias.
- `logs/`: logs de geracao e validacao.

Se algo em `build/` for necessario para reproduzir uma publicacao, registre o comando em `README.md` ou `docs/VALIDATION.md`.
