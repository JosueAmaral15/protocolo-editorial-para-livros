# Scripts

Use esta pasta para automacoes reprodutiveis do projeto.

Exemplos:

- conversao de Markdown para DOCX, EPUB e PDF;
- validacao de artefatos;
- auditoria de termos proibidos;
- extracao de texto;
- insercao de QR code;
- geracao de metadados.

Todo script importante deve ter um comando de uso registrado no `README.md` da raiz ou em `docs/VALIDATION.md`.
