# Capas

Guarde aqui somente arquivos de arte de capa deste livro.

- arte-base sem texto e suas fontes;
- capa frontal aprovada;
- capa completa limpa: contracapa + lombada + capa;
- preview separado com guias, sangria e area segura;
- versao com orelhas somente quando a grafica fornecer template compativel;
- metadados, prompt, hashes e notas de revisao.

Nao use um wireframe tecnico como capa comercial. Confirme a largura oficial da
lombada depois que a plataforma processar o miolo final. Para UICLAP, salve o
gabarito/SVG gerado pelo Portal neste projeto e use-o como referencia final; as
medidas de um piloto anterior servem apenas para planejamento.
