# Manuscrito

Use esta pasta para o texto-fonte do livro.

- `original/`: arquivo original preservado sem modificacoes.
- `trabalho/`: manuscrito ativo em edicao.
- `revisado/`: versoes revisadas prontas para prova.
- `final/`: fonte final aprovada antes da geracao de artefatos.

Regra: nao trate PDF, DOCX ou EPUB como fonte principal se houver Markdown ou outro manuscrito editavel mais confiavel.
