# DECISIONS - Projeto Romance Com Orelhas

## Usar capa tecnica didatica

**Status:** aceito para exemplo.

**Contexto:** o repositorio precisa demonstrar a diferenca entre capa frontal, capa completa e capa com orelhas sem depender de arte comercial final.

**Decisao:** gerar uma capa tecnica deterministica com Pillow.

**Impacto:** o exemplo e reprodutivel e leve, mas nao deve ser confundido com capa final de romance.
