# Historico - Projeto Romance Com Orelhas

## Registro inicial

Este e um projeto exemplo do repositorio `protocolo-editorial-para-livros`.

Objetivo:

- demonstrar estrutura editorial para romance;
- gerar capa frontal;
- gerar capa completa com contracapa, lombada e capa;
- gerar capa completa com orelhas;
- documentar metadados e validacao.

Nenhum conteudo aqui deve ser tratado como livro final.
