# DECISIONS - Projeto Livro Academico UICLAP

## Usar capa tecnica didatica

**Status:** aceito para exemplo.

**Contexto:** o repositorio precisa mostrar a estrutura de capas sem depender de imagem comercial, servico externo ou arquivo pesado.

**Decisao:** gerar uma capa tecnica deterministica com Pillow.

**Impacto:** o exemplo e reprodutivel e leve, mas nao deve ser confundido com capa final de venda.
