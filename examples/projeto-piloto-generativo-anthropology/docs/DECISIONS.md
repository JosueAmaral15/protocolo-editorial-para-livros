# DECISIONS - Projeto Piloto Generativo Anthropology

## Usar piloto generativo real

**Status:** aceito

**Contexto:** exemplos geometricos por script demonstram medidas, mas nao demonstram qualidade visual de capa comercial.

**Decisao:** incluir o piloto generativo `en-anthropology` como exemplo pratico principal.

**Impacto:** o repositorio passa a ter um exemplo visual concreto de arte-base sem texto, tipografia local e capa completa externa.
