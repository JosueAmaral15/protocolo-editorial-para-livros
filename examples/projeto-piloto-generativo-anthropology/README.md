# Projeto Piloto Generativo Anthropology

Este e o exemplo pratico correto para demonstrar uma capa em padrao mais proximo de capa comercial: arte-base generativa sem texto, tipografia local e capa completa externa separada do miolo.

Ele foi copiado do piloto aprovado em:

```text
/home/josue/Documents/josue-writter-workspace/books/capas/pilotos-generativos/en-anthropology/
```

## Arquivos Principais

- `assets/capas/en-anthropology-base-generativa-v1.png`: arte-base generativa sem texto.
- `publicacao/capas/en-anthropology-capa-frontal-piloto-v2.png`: capa frontal com tipografia local.
- `publicacao/capas/en-anthropology-capa-completa-uiclap-piloto-v2.jpg`: capa completa limpa, com contracapa, lombada e capa.
- `publicacao/capas/en-anthropology-capa-completa-uiclap-piloto-v2-preview-validacao.jpg`: preview com guias.
- `docs/prompt-base-generativa.md`: prompt usado para a arte-base.
- `scripts/build_pilot_cover.py`: script de composicao local.
- `publicacao/capas/metadata-v2.json`: metadados, dimensoes, hashes e estado do piloto.

## Status

`piloto_generativo_revisavel_nao_final`

Este exemplo demonstra uma direcao visual comercial, enquanto os wireframes
tecnicos demonstram apenas montagem de paineis. Ainda assim, nao deve ser
tratado como capa final de upload sem revisao humana, confirmacao de lombada
oficial e validacao na plataforma.
